- [Марс](https://libreno.github.io/layout-designer/mars.html)

Выполнена на Yandex Cup 2020 по Frontend.

Текст задания:

*Марсианская археологическая экспедиция совершила потрясающее открытие: во время раскопок в долине Маринера были найдены таблички с изображениями других планет. Как утверждает глава экспедиции, это подтверждает гипотезу о том, что древние марсиане знали о космосе и наблюдали за небом.
Такую сенсацию необходимо срочно передать на Землю для вечернего выпуска археологических новостей. Но чтобы переслать фотографии табличек, у экспедиции в походном наборе есть только радиопередатчик с пропускной способностью в 1 байт в секунду. В бинарном виде фотографии занимают слишком много места, чтобы успеть переслать их на Землю к выпуску новостей, который начнётся уже через 25 минут.
Глава экспедиции просит вас помочь сократить размер фотографий так, чтобы успеть сообщить жителям Земли о сенсационном открытии как можно раньше.*

*Примечания*

*Вам нужно сверстать картинку 512 на 512 пикселей при помощи HTML и CSS. Весь код должен уместиться в 1024 символа с учётом переносов строк и пробелов. Использовать SVG, встроенные изображения и JavaScript нельзя.*

\<style\>  
    /* стили */  
\</style\>  
\<body\>  
    \<!-- верстка --\>\
\</body\>

*У вас есть походный инструмент для эффективного сжатия изображений: [https://yadi.sk/d/TE2xicoAatJZMg](https://yadi.sk/d/TE2xicoAatJZMg)*

*Вёрстка должна в точности соответствовать изображению (pixel perfect).
Для рисования карты использовались следующие цвета: #1d057f, #ffd21f, #f7c407.*
